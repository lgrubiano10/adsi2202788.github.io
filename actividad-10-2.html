La etiqueta video: Para especificar la ubicación del archivo que se reproducirá, puede usar el elemento <source>de apertura dentro del elemento <video>. (Esto debe reemplazar el atributo src en la etiqueta <video> de apertura.) También puede utilizar varios elementos <source> para especificar que el video está disponible en varios formatos. (Debido a un error en el Ipad, debe proporcionar el video como MP4 en el primer formato, de lo contrario puede que no inicie).

src: Este atributo especifica la ruta de acceso al video.

type: Se debe utilizareste atributo para decirle al navergador qué formato es el video. De lo contrario, descargará parte del video para ver si puede reproducir el archivo. (que tomará tiempo y ancho de banda).

codecs: Elcodecque se usó para codificar el video se suministra dentro del atributo type. Tenga en cuenta el uso de comillas simples, así comocomillas dobles en el atributo de tipo, cuando se suministra.
