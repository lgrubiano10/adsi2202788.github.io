Cada elemento HTML puede llevar el atributo id. Se utiliza para identificar de forma unívoca a ese elemento de otros elementos de la página. Su valor debe comenzar con una letra o un guión bajo (no un número o cualquier otro carácter). Es importante para que no haya dos elementos en la misma página que tengan el mismo valor para sus atributos ID (de lo contrario el valor ya no es único).

Dar a un elemento una identidad única, le permite darle un estilo diferente que cualquier otra instancia del mismo elemento de la página

Por ejemplo, es posible que desee asignar, a un párrafo dentro de la página (quizás un párrafo que contiene una cotización de extracción), un estilo diferente que todos los demás párrafos. En el ejemplo, elpárrafo con el atributo ID cuyo valor es pullquote, se hace en mayúsuculas usando código CSS.
