En HTML5 no es necesario proporcionar valores para todos los atributos, como los controles, reproducción automática y atributos de bucle utilizados con el elemento <video>. Estos atributos son como interruptores de encendido/apagado. Si el atributo está presente,activa esa opción. Si se omite el atributo la opción queda desactivada. Si el navegador no admite el elemento <video> o el formato de video utilizado, se mostrará lo que está entre la etiqueta <video> de apertura y la etiqueta de </video> de cierre.

preload:Este atributo le indica al navegador qué hacer cuando se carga la página. Puede tener uno de tres valores:
none: El navegador no debe cargar el video hasta que el usuario presiona Play.
auto: El navegador debe descargar el video cuando se carga la página.
metadata: El navegador debe simplemente recopilar información como el tamaño, el primer fotograma, la lista de pistas y la duración.
