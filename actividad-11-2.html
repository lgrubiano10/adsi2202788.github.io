Entendiendo los CSS: Inspirados en los cuadros

La clave para entender cómo funciona CSS es imaginar que hay un cuadro invisible alrededor de cada elemento HTML.

En la primera página, se puede ver una página HTML básica.

En la segunda página, se puede ver la misma página HTML, pero se han agregado contornos a cada uno de los elementos para que pueda ver cómo los CSS tratarán cada elemento como si estuvieran dentro de su propio cuadro.
