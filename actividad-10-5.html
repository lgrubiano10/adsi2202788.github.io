Youtube video ID - Youtube mostrará un ID (como EF81Pb5IWoU), al guardar (o reproducir) un video. Puede utilizar este ID y hacer referencia a su video en el código HTML.

Reproducción de un video de Youtube en HTML:
Para reproducir un video en una página web haga lo siguiente:

  • Sube el video a Youtube
  • Anota el ID del video
  • Define un elemento <iframe> en tu página web
  • Deja que el atributo src apunte a la URL del video
  • Utiliza los atributos width y height para especificar la dimensión del reproductor
