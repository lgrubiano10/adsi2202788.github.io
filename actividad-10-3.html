<audio> HTML5 introdujo el elemento <audio> para incluir archivos de audio en sus páginas. Al igual que con el video HTML5,los navegadores esperan diferentes formatos para el audio. El elemento <audio> tiene un número de atributos que le permiten controlar la reproducción del audio:

src: Este atributo especifica la ruta de acceso al archivo de audio.

controls: Este atributo indica si el reproductor debe mostrar los controles. Si no utiliza este atributo, no se mostrarán controles de forma predetermianda. También puede especificar sus propios controles mediante JavScript.

autoplay: Lapresencia de este atributo indica que elaudio debe empezar a reproducirse automáticamente. (Se considera una mejor práctica permitir que los visitantes elijan reproducir audio.)
